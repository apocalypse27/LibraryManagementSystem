-The Application is built on Eclipse (4.7.0 Oxygen Release)
-Built using JRE JavaSE-1.8 
-SceneBuilder 2.0 was included in the Eclipse. Follow instructions at 
http://o7planning.org/en/10621/install-javafx-scene-builder-into-eclipse to include SceneBuilder into eclipse IDE.
-mysql-connector-java-5.1.44-bin is the JRE used.
Follow instructions at https://stackoverflow.com/questions/13442747/set-the-classpath-in-order-to-connect-between-mysql-to-jdbc to include JDBC to Java's class path in Eclipse
-MySQL 5.7 is used.
